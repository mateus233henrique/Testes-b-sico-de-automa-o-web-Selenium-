# Automação de Testes com Selenium

Este projeto contém testes automatizados básicos utilizando Python e Selenium.

## Objetivo
Praticar automação de testes web, simulando ações do usuário e validando comportamentos da aplicação.

## Tecnologias utilizadas
- Python
- Selenium WebDriver

## Cenário automatizado
- Acesso ao site SauceDemo
- Login com usuário válido
- Verificação de carregamento da página de produtos

## O que está sendo validado
- Elementos localizados corretamente (find_element)
- Interação com inputs e botões

## Aprendizados
- Uso de locators (ID, CLASS_NAME)
- Diferença entre find_element e find_elements
- Importância de waits para sincronização
